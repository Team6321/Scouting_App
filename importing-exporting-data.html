Importing Data

If you wish to import all data (season configuration, event data, and team data) for one or more seasons, check those seasons, choose a file and press Import. This will add to/overwrite any previously saved elements/pit scouting questions for those seasons.
Please make sure that the input file ends in .rcubedscoutconfig.


Exporting Data

If you wish to export the current configuration data for one or more seasons to an external file (ending in .rcubedscoutconfig), check those seasons and click the following button.

Export Data
