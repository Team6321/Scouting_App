Season Configuration



Enter the name of a new season and/or select a season to continue to entering season specific data.







Enter in any scoring opportunities in the first text box and how many points that scoring opportunity is worth in the second text box.

Also, please enter the type of answer to be expected when answering questions about the scoring opportunity in the Match Data section of Team Data.
For example, if the scoring opportunity was 'ball,' the input type would probably be numeric.
However, if the scoring opportunity was 'endgame climb,' the input type would probably be text ('yes the team can', 'no the team can't').
