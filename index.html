Welcome to the Rouse Robotics Scouting Web App!



Click "Navigation Menu" to see the different parts of the site.



● Go to the Season Configuration page to enter custom data about this season, such as game elements and pit scouting questions.

● Go to the Event Data page to enter events and teams attending that event.

● Go to the Team Data page to enter and view team data such as pit scouting and match info for a given event.

● Go to the Importing and Exporting Data page to import/export currently stored data in the Season Configuration page, the Event Data page, and the Team Data page.
